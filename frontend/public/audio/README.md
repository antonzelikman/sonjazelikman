# Sonja's Adventures — Audio Files

Place your 4 MP3 files here. Each tab plays its own track with crossfade.

## Required files:
- you.mp3         → Tab 1: YOU  (current: "Bara Få Va Mig Själv" ✅)
- gift.mp3        → Tab 2: YOUR GIFT
- things.mp3      → Tab 3: THINGS YOU NEED
- adventures.mp3  → Tab 4: ADVENTURES

## Notes:
- Files must be in MP3 format
- Crossfade duration: 1.5 seconds
- Audio requires user interaction to start (click the music button)
- If a file is missing, that tab will play in silence
